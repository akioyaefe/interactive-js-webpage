# Interactive JS Webpage

**Author:** Efemena Akioya

## Description
This project is a dynamic web page demonstrating JavaScript fundamentals including event handling, interactive UI elements, and custom form validation. Users can interact with buttons, inputs, and forms while exploring features like light/dark mode, counters, collapsible FAQs, and tabbed interfaces.

## Features

### Part 1: Event Handling
- Click, mouseover, and keyboard input events
- Dynamic responses to user actions

### Part 2: Interactive Elements
- Light/Dark mode toggle
- Counter button
- Collapsible FAQ section
- Tabbed interface for content switching

### Part 3: Form Validation
- Custom form validation for name, email, and password fields
- User-friendly error messages
- Prevents form submission if validation fails

## Files
- `index.html` – The main web page structure
- `script.js` – JavaScript file handling events, interactive features, and form validation
- `style.css` – Optional CSS file for styling the page

## Usage
1. Clone or download the repository.
2. Open `index.html` in a web browser.
3. Interact with buttons, forms, and UI elements to see JavaScript in action.
